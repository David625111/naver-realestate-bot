## 🔍 각 필터 설명 (아주 쉽게!)

### 1️⃣ **property_types** - 집 종류

**뭘 하는 건가요?**
- 아파트만 볼지, 오피스텔만 볼지 정해요

**선택할 수 있는 것:**
```json
["APT"]           → 아파트만
["OPST"]          → 오피스텔만
["APT", "OPST"]   → 둘 다 (지금 설정)
```

**예시:**
```json
"property_types": ["APT"]  ← 아파트만 보고 싶을 때
```

---

### 2️⃣ **trade_types** - 거래 방식

**뭘 하는 건가요?**
- 집을 사고 싶은지, 전세인지, 월세인지 정해요

**선택할 수 있는 것:**
```json
"A1"  → 매매 (집을 사는 거)
"B1"  → 전세
"B2"  → 월세
"B3"  → 단기임대
```

**예시:**
```json
"trade_types": ["A1"]           ← 매매만
"trade_types": ["B1"]           ← 전세만
"trade_types": ["A1", "B1"]     ← 매매랑 전세 둘 다 (지금 설정)
"trade_types": ["A1", "B1", "B2"]  ← 매매, 전세, 월세 전부
```

---

### 3️⃣ **price_range** - 가격 범위 💰

**뭘 하는 건가요?**
- 얼마부터 얼마까지의 집을 볼지 정해요
- **단위: 만원** (10,000원 = 1만원)

**예시 계산:**
```
50000 만원 = 5억원
100000 만원 = 10억원
150000 만원 = 15억원
```

**현재 설정:**
```json
"price_range": {
  "A1": {"min": 50000, "max": 150000},  ← 매매: 5억~15억
  "B1": {"min": 30000, "max": 80000}    ← 전세: 3억~8억
}
```

**바꾸는 방법:**

**예시 1: 매매 10억~20억**
```json
"price_range": {
  "A1": {"min": 100000, "max": 200000},
  "B1": {"min": 30000, "max": 80000}
}
```

**예시 2: 전세 5억~10억**
```json
"price_range": {
  "A1": {"min": 50000, "max": 150000},
  "B1": {"min": 50000, "max": 100000}
}
```

**예시 3: 가격 제한 없애기 (모든 가격 다 보기)**
```json
"price_range": {
  "A1": {"min": 0, "max": 999999},
  "B1": {"min": 0, "max": 999999}
}
```

---

### 4️⃣ **area_range** - 집 크기 📐

**뭘 하는 건가요?**
- 집이 얼마나 큰지 정해요
- **단위: ㎡ (제곱미터)**

**평수로 환산:**
```
59㎡ ≈ 18평
84㎡ ≈ 25평
99㎡ ≈ 30평
```

**현재 설정:**
```json
"area_range": {"min": 59, "max": 84}  ← 18평~25평
```

**바꾸는 방법:**

**예시 1: 30평~40평 (99㎡~132㎡)**
```json
"area_range": {"min": 99, "max": 132}
```

**예시 2: 크기 상관없이 다 보기**
```json
"area_range": {"min": 0, "max": 999999}
```

**평수 → ㎡ 변환표:**
```
10평 = 33㎡
15평 = 50㎡
20평 = 66㎡
25평 = 84㎡
30평 = 99㎡
35평 = 116㎡
40평 = 132㎡
```

### 5️⃣ **approval_year** - 건물 나이 🏢

**뭘 하는 건가요?**
- 언제 지어진 집을 볼지 정해요
- 너무 오래된 집은 안 보고 싶을 때 써요

**현재 설정:**
```json
"approval_year": {"min": 2018, "max": 2024}  ← 2018년~2024년 사이
```

**바꾸는 방법:**

**예시 1: 5년 이내 새 집만 (2021년~2026년)**
```json
"approval_year": {"min": 2021, "max": 2026}
```

**예시 2: 10년 이내 (2016년~2026년)**
```json
"approval_year": {"min": 2016, "max": 2026}
```

**예시 3: 건물 나이 상관없이 다 보기**
```json
"approval_year": {"min": 0, "max": 9999}
```

### 6️⃣ **household_count** - 세대수 🏘️

**뭘 하는 건가요?**
- 아파트 단지에 집이 몇 개 있는지
- 큰 단지가 좋은지, 작은 단지가 좋은지 정해요

**현재 설정:**
```json
"household_count": {"min": 300, "max": 2000}  ← 300세대~2000세대
```

**바꾸는 방법:**

**예시 1: 대단지만 (1000세대 이상)**
```json
"household_count": {"min": 1000, "max": 9999}
```

**예시 2: 소규모 단지 (100~500세대)**
```json
"household_count": {"min": 100, "max": 500}
```

**예시 3: 세대수 상관없이**
```json

```

---

### 7️⃣ **floor_types** - 층수 🏢

**뭘 하는 건가요?**
- 몇 층에 있는 집을 볼지 정해요

**선택할 수 있는 것:**
```
"1층"     → 1층만
"저층"    → 아래쪽 (2층~전체 1/3)
"중간층"  → 중간 (전체 1/3~2/3)
"고층"    → 위쪽 (전체 2/3~최상층-1)
"탑층"    → 꼭대기 층
```

**예시:**
- 25층 건물이라면:
  - 저층: 2~8층
  - 중간층: 9~16층
  - 고층: 17~24층
  - 탑층: 25층

**현재 설정:**
```json
"floor_types": ["중간층", "고층"]  ← 중간층이랑 고층만
```

**바꾸는 방법:**

**예시 1: 고층만**
```json
"floor_types": ["고층"]
```

**예시 2: 1층 빼고 다**
```json
"floor_types": ["저층", "중간층", "고층", "탑층"]
```

**예시 3: 층수 상관없이 다 보기**
```json
"floor_types": []  ← 비워두기!
```

### 🔟 **directions** - 집 방향 🧭

**뭘 하는 건가요?**
- 집이 어느 쪽을 향하는지 정해요
- 햇빛이 잘 드는지 중요할 때 써요

**선택할 수 있는 것:**
```
"남향"    → 남쪽 (제일 밝아요!)
"남동향"  → 남쪽+동쪽
"남서향"  → 남쪽+서쪽
"동향"    → 동쪽 (아침 햇빛)
"서향"    → 서쪽 (오후 햇빛)
"북향"    → 북쪽 (어두워요)
```

**현재 설정:**
```json
"directions": ["남향", "남동향", "남서향"]  ← 남쪽 방향만
```

**바꾸는 방법:**

**예시 1: 남향만**
```json
"directions": ["남향"]
```

**예시 2: 남향, 동향**
```json
"directions": ["남향", "동향"]
```

**예시 3: 방향 상관없이**
```json
"directions": []  ← 비워두기!
```

### 1️⃣1️⃣ **loan** - 융자금 💳

**뭘 하는 건가요?**
- 집에 빚(대출)이 있는지 없는지

**선택할 수 있는 것:**
```
"상관없음"        → 빚 있어도 상관없어
"융자금 없음"     → 빚 없는 집만
"융자금30%미만"   → 빚이 집값의 30%보다 적은 것만
```

**현재 설정:**
```json
"loan": "상관없음"  ← 빚 있어도 괜찮아
```

**바꾸는 방법:**

**예시 1: 빚 없는 집만**
```json
"loan": "융자금 없음"
```

**예시 2: 빚이 적은 집만 (30% 미만)**
```json
"loan": "융자금30%미만"
```

## ⚠️ 주의할 점

### 1. 쉼표(,) 조심!

**❌ 잘못된 예:**
```json
{
  "room_count": [2, 3]  ← 여기 쉼표 있어야 해요!
  "bathroom_count": [2]
}
```

**✅ 올바른 예:**
```json
{
  "room_count": [2, 3],  ← 쉼표 있음!
  "bathroom_count": [2]
}
```

### 2. 따옴표 조심!

**❌ 잘못된 예:**
```json
"directions": [남향, 동향]  ← 따옴표 없음!
```

**✅ 올바른 예:**
```json
"directions": ["남향", "동향"]  ← 따옴표 있음!
```

### 3. 대괄호 [ ] 조심!

**리스트(여러 개 선택):**
```json
"room_count": [2, 3, 4]  ← 대괄호로 감싸기
```

**범위(최소~최대):**
```json
"price_range": {"min": 50000, "max": 100000}  ← 중괄호 사용
```

## 🆘 자주 묻는 질문

### Q: 필터를 바꿨는데 적용이 안 돼요!

**A:** 파일을 저장했는지 확인하세요!
1. 메모장에서 `Ctrl + S`
2. 파일 닫기
3. 프로그램 다시 실행

### Q: 매물이 하나도 안 나와요!

**A:** 필터가 너무 까다로워요! 이렇게 해보세요:
1. 가격 범위를 넓히기
2. 층수, 방향 등을 `[]`로 비우기
3. 지역 코드가 맞는지 확인

### Q: 매물이 너무 많이 나와요!

**A:** 필터를 더 엄격하게!
1. 가격 범위 좁히기
2. 층수, 방향, 방 개수 정하기
3. 건물 나이 제한하기

### Q: 파일을 잘못 수정했어요!

**A:** 원래대로 돌리기:
1. `config/filters.json` 파일 열기
2. 전부 지우기
3. 이 가이드의 "템플릿 1" 복사해서 붙여넣기
4. 저장하기
