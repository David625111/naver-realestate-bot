## 2단계: Python 설치하기 (5분)

> **Python이 뭐예요?** 🐍
> 
> 컴퓨터 언어예요! 우리가 한글로 말하듯이, 컴퓨터는 Python이라는 언어로 말해요.
> Python을 설치하면 컴퓨터가 우리가 만든 프로그램을 이해할 수 있어요.

### 설치 방법:

**1️⃣ 다운로드 사이트 가기**
- 인터넷 브라우저(크롬, 엣지 등)를 켜세요
- 주소창에 입력: `https://www.python.org/downloads/`
- 엔터를 누르세요

**2️⃣ 다운로드 버튼 클릭**
- 노란색 "Download Python" 버튼을 찾으세요
- 버튼을 클릭하세요
- 파일이 다운로드됩니다 (1~2분 걸려요)

**3️⃣ 설치 시작**
- 다운로드한 파일을 찾아서 더블클릭하세요
- **⚠️ 중요!** 아래쪽에 있는 체크박스 클릭:
  ```
  ☑ Add Python to PATH
  ```
  이걸 꼭 체크해야 해요! (이게 없으면 나중에 안 돼요)
  
- "Install Now" 버튼 클릭
- 잠깐 기다리면 설치가 끝나요 (3~5분)

**4️⃣ 설치 확인하기**

Windows 검색창에서 "PowerShell" 입력하고 열기
(파란색 화면이 나와요)

이렇게 입력하고 엔터:
```
python --version
```

**✅ 성공:** `Python 3.11.0` 같은 글자가 나오면 성공!
**❌ 실패:** 에러가 나오면 다시 설치하세요 (체크박스 확인!)

## 3단계: Git 설치하기 (5분)

> **Git이 뭐예요?** 📦
> 
> 프로그램 코드를 내려받는 도구예요.
> 마치 우체부가 택배를 가져다 주는 것처럼, Git은 코드를 가져다 줘요.

### 설치 방법:

**1️⃣ 다운로드**
- 주소: `https://git-scm.com/downloads`
- "Download for Windows" 클릭

**2️⃣ 설치**
- 다운로드한 파일 더블클릭
- 계속 "Next" 버튼만 클릭하세요 (설정 건드릴 필요 없어요)
- "Install" 클릭
- 완료되면 "Finish" 클릭

**3️⃣ 확인**

PowerShell에서 입력:
```
git --version
```

**✅ 성공:** `git version 2.40.0` 같은 글자가 나오면 성공!

## 4단계: 텔레그램 봇 만들기 (10분)

> **봇이 뭐예요?** 🤖
> 
> 자동으로 메시지를 보내주는 로봇이에요!
> 우리가 만든 프로그램이 이 봇을 통해서 휴대폰으로 알림을 보낼 거예요.

### Part A: 텔레그램 앱 설치

**1️⃣ 앱 다운로드**
- 휴대폰에서 앱스토어(아이폰) 또는 플레이스토어(안드로이드) 열기
- "텔레그램" 검색
- 파란색 종이비행기 아이콘의 "Telegram" 설치
- 앱 열고 전화번호로 가입

### Part B: 봇 만들기

**1️⃣ BotFather 찾기**
- 텔레그램 앱에서 상단 검색창 클릭
- `@BotFather` 입력
- **공식 BotFather** 선택 (파란색 체크마크가 있어요)
- "START" 버튼 클릭

**2️⃣ 새 봇 만들기**

다음 메시지를 순서대로 입력하세요:

```
/newbot
```
엔터를 누르면 BotFather가 질문해요:

**질문 1:** "봇 이름이 뭐예요?"
```
내 부동산 알림봇
```
(원하는 이름으로 바꿔도 돼요!)

**질문 2:** "봇 사용자명은요?"
```
my_house_finder_bot
```
(영어로만, bot으로 끝나야 해요. 다른 사람이 이미 쓰고 있으면 안 돼요)

**3️⃣ 토큰 저장하기** 🔑

BotFather가 이렇게 말해줘요:
```
Done! Congratulations on your new bot.
Use this token to access the HTTP API:
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

⚠️ **이 숫자와 글자를 복사해서 메모장에 저장하세요!**

이게 바로 **토큰**이에요. (마치 집 열쇠 같은 거예요!)
- 길게 누르면 복사할 수 있어요
- 메모장에 붙여넣기 하세요
- "텔레그램 토큰"이라고 제목 써두세요

### Part C: 채팅 ID 알아내기

**1️⃣ 봇에게 메시지 보내기**
- 검색창에 방금 만든 봇 이름 입력 (예: `@my_house_finder_bot`)
- 봇 클릭
- "START" 버튼 클릭
- 아무 메시지나 보내기 (예: "안녕!")

**2️⃣ 채팅 ID 찾기**

컴퓨터로 인터넷 브라우저를 켜세요.

주소창에 이렇게 입력하세요 (토큰 부분에 아까 복사한 토큰을 붙여넣으세요):
```
https://api.telegram.org/bot1234567890:ABCdefGHIjklMNOpqrsTUVwxyz/getUpdates
```

**예시:** 토큰이 `9876543210:XYZabcDEFuvw`라면:
```
https://api.telegram.org/bot9876543210:XYZabcDEFuvw/getUpdates
```

엔터를 누르면 이런 글자들이 나와요:
```json
{
  "ok": true,
  "result": [
    {
      "message": {
        "chat": {
          "id": 987654321,    ← 이 숫자를 찾으세요!
        }
      }
    }
  ]
}
```

**3️⃣ 숫자 저장하기**

`"id":` 다음에 나오는 숫자를 복사하세요 (예: `987654321`)
- 메모장에 저장하세요
- "채팅 ID"라고 제목 써두세요

**📝 지금까지 저장한 것:**
```
텔레그램 토큰: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
채팅 ID: 987654321
```

## 5단계: 내가 원하는 지역 찾기 (5분)

> **왜 지역 코드가 필요해요?** 🗺️
> 
> 우리가 집을 찾고 싶은 동네를 컴퓨터에게 알려줘야 해요.
> 컴퓨터는 "강남구 대치동"이라고 말해도 못 알아듣고, 숫자로 말해야 알아들어요.

### 지역 코드 찾는 방법:

**1️⃣ 네이버 부동산 가기**
- 인터넷 브라우저 열기
- 주소창에 입력: `https://new.land.naver.com`
- 엔터

**2️⃣ 원하는 동네 검색**
- 화면 위에 검색창이 있어요
- 원하는 동네 입력 (예: "강남구 대치동")
- 엔터 또는 검색 버튼 클릭

**3️⃣ 주소창에서 숫자 찾기**

주소창을 보면 이렇게 길게 나와요:
```
https://new.land.naver.com/complexes?cortarNo=1168010600&tradTpCd=...
```

`cortarNo=` 다음에 나오는 숫자를 찾으세요!
```
cortarNo=1168010600
          ^^^^^^^^^^
          이 숫자!
```

**4️⃣ 숫자 저장하기**
- 숫자 10개를 복사하세요 (예: `1168010600`)
- 메모장에 저장하세요
- "지역 코드"라고 제목 써두세요

**여러 동네를 보고 싶다면?**
- 다른 동네도 같은 방법으로 찾아서 저장하세요
- 쉼표로 구분해서 적으세요
- 예: `1168010600,1165010100,1168010700`

**📝 유명한 동네 코드:**
```
1168010600 - 강남구 대치동
1165010100 - 서초구 반포동
1168010700 - 강남구 도곡동
1168011000 - 강남구 삼성동
1111010100 - 종로구 청운효자동
```

## 6단계: 컴퓨터에 프로그램 준비하기 (10분)

> **이제 뭘 하는 거예요?** 💻
> 
> 우리가 쓸 프로그램을 컴퓨터에 내려받고,
> 프로그램이 일할 수 있는 환경을 만들어줄 거예요.

### Part A: PowerShell 열기

**1️⃣ PowerShell 찾기**
- 윈도우 시작 버튼 클릭 (왼쪽 아래 창문 모양)
- "PowerShell" 입력
- "Windows PowerShell" 클릭
- 파란색 화면이 나타나요

### Part B: 프로그램 폴더로 이동

PowerShell에 이렇게 입력하세요:
```powershell
cd "c:\Users\BYEONGWOOK MIN\naver-realestate-bot"
```

⚠️ **이름이 다르면:**
- `BYEONGWOOK MIN` 부분을 자기 이름으로 바꾸세요
- 예: `cd "c:\Users\김철수\naver-realestate-bot"`

엔터를 누르세요!

### Part C: 작업 공간 만들기

> **작업 공간이 뭐예요?** 🏠
> 
> 프로그램이 일할 때 필요한 도구들을 넣어두는 방이에요.
> 마치 책상 위에 공부할 때 필요한 것들을 올려두는 것처럼요!

**1️⃣ 작업 공간 만들기**

PowerShell에 입력:
```powershell
python -m venv venv
```
엔터를 누르고 30초 정도 기다리세요.

**2️⃣ 작업 공간 켜기**

PowerShell에 입력:
```powershell
.\venv\Scripts\Activate.ps1
```

**❌ 에러가 나면?**

이렇게 입력하고 다시 시도:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
"Y"를 입력하고 엔터
그리고 다시 위의 명령어 입력

**✅ 성공하면:**
앞에 `(venv)`라는 글자가 나와요!
```
(venv) PS C:\Users\...>
```

### Part D: 필요한 도구들 설치하기

> **도구가 뭐예요?** 🔧
> 
> 프로그램이 일하려면 여러 가지 도움이 필요해요.
> 네이버에서 정보 가져오기, 텔레그램으로 메시지 보내기 등등...
> 이런 걸 도와주는 도구들을 설치할 거예요.

PowerShell에 입력:
```powershell
pip install -r requirements.txt
```

- 엔터를 누르고 2~3분 기다리세요
- 많은 글자들이 막 지나가요 (정상이에요!)
- "Successfully installed"라는 글자가 나오면 끝!

### Part E: 설정 파일 만들기

> **설정 파일이 뭐예요?** 📝
> 
> 프로그램에게 "이렇게 해!"라고 알려주는 쪽지예요.
> 우리가 아까 저장했던 토큰, 채팅 ID, 지역 코드를 여기에 써줄 거예요.

**1️⃣ 설정 파일 만들기**

PowerShell에 이렇게 입력하세요:
```powershell
@"
TELEGRAM_BOT_TOKEN=여기에_토큰_붙여넣기
TELEGRAM_CHAT_ID=여기에_채팅ID_붙여넣기
SEARCH_REGIONS=여기에_지역코드_붙여넣기
TRADE_TYPES=A1,B1
"@ | Out-File -FilePath .env -Encoding UTF8
```

엔터를 누르세요.

**2️⃣ 설정 파일 열어서 수정하기**

PowerShell에 입력:
```powershell
notepad .env
```

메모장이 열려요!

**3️⃣ 진짜 정보로 바꾸기**

메모장에서 이렇게 바꾸세요:

**바꾸기 전:**
```
TELEGRAM_BOT_TOKEN=여기에_토큰_붙여넣기
TELEGRAM_CHAT_ID=여기에_채팅ID_붙여넣기
SEARCH_REGIONS=여기에_지역코드_붙여넣기
TRADE_TYPES=A1,B1
```

**바꾼 후 (예시):**
```
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=987654321
SEARCH_REGIONS=1168010600
TRADE_TYPES=A1,B1
```

⚠️ **주의할 것:**
- 따옴표 없이 그냥 붙여넣기
- 등호(=) 앞뒤로 공백 없이
- 실제 숫자와 글자로 바꾸기

**4️⃣ 저장하기**
- "파일" → "저장" 클릭
- 메모장 닫기

**📝 각 줄이 뭐예요?**
```
TELEGRAM_BOT_TOKEN    → 텔레그램 봇 열쇠
TELEGRAM_CHAT_ID      → 내 전화번호 (텔레그램용)
SEARCH_REGIONS        → 찾을 동네
TRADE_TYPES           → A1=매매, B1=전세, B2=월세
```

## 🎯 자주 묻는 질문 (FAQ)

### Q1: 에러가 나요! 😭

**A:** 가장 흔한 실수들:

1. **PowerShell 실행 정책 오류**
   ```powershell
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
   
2. **가상환경 안 켜짐**
   - 앞에 `(venv)` 있는지 확인
   - 없으면 `.\venv\Scripts\Activate.ps1` 다시 실행

3. **텔레그램 메시지 안 옴**
   - 봇에게 "START" 눌렀는지 확인
   - `.env` 파일 다시 확인

### Q2: 집이 너무 많이 나와요!

**A:** 필터를 조정하세요!

메모장으로 이 파일을 여세요:
```
c:\Users\BYEONGWOOK MIN\naver-realestate-bot\config\filters.json
```

여기서 원하는 조건을 바꿀 수 있어요:
- 가격 범위
- 집 크기
- 방 개수
- 등등...

### Q3: 집이 하나도 안 나와요!

**A:** 필터가 너무 까다로워요!

`filters.json` 파일에서:
- 가격 범위를 넓히세요
- 다른 조건들을 지우세요

### Q4: 프로그램을 다시 실행하려면?

**A:** 
1. PowerShell 열기
2. `cd "c:\Users\BYEONGWOOK MIN\naver-realestate-bot"` 입력
3. `.\venv\Scripts\Activate.ps1` 입력
4. `cd src` 입력
5. `python main.py` 입력

### Q5: 프로그램을 멈추려면?

**A:** `Ctrl + C` (컨트롤 키와 C를 동시에 누르기)
