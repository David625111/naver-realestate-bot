# 🎨 그림으로 보는 설치 가이드

**한눈에 보는 전체 과정!** 👀

---

## 📊 전체 흐름도

```
시작! 🚀
  │
  ├─→ [1단계] 준비물 확인 ✅
  │   ├─ 컴퓨터 있나요? YES!
  │   ├─ 인터넷 되나요? YES!
  │   └─ 휴대폰 있나요? YES!
  │
  ├─→ [2단계] Python 설치 🐍
  │   ├─ python.org 방문
  │   ├─ 다운로드
  │   └─ 설치 (체크박스 클릭!)
  │
  ├─→ [3단계] Git 설치 📦
  │   ├─ git-scm.com 방문
  │   ├─ 다운로드
  │   └─ 설치 (Next만 클릭)
  │
  ├─→ [4단계] 텔레그램 봇 만들기 🤖
  │   ├─ @BotFather 찾기
  │   ├─ /newbot 입력
  │   ├─ 토큰 받기 🔑
  │   └─ 채팅 ID 확인 🆔
  │
  ├─→ [5단계] 지역 코드 찾기 🗺️
  │   ├─ 네이버 부동산 접속
  │   ├─ 동네 검색
  │   └─ URL에서 숫자 복사
  │
  ├─→ [6단계] 프로그램 준비 💻
  │   ├─ PowerShell 열기
  │   ├─ 가상환경 만들기
  │   ├─ 도구 설치
  │   └─ .env 파일 만들기
  │
  ├─→ [7단계] 테스트! 🎯
  │   ├─ 기본 테스트
  │   ├─ 텔레그램 테스트
  │   └─ 실제 실행
  │
  └─→ 성공! 🎉
```

---

## 🎯 각 단계별 목표

### 1단계: 준비물 확인
```
┌─────────────────┐
│  컴퓨터 + 인터넷  │
│   + 휴대폰      │
└─────────────────┘
       ↓
    ✅ 준비 완료!
```

### 2단계: Python 설치
```
python.org
    ↓
  다운로드
    ↓
  설치 파일 실행
    ↓
[☑] Add Python to PATH  ← 꼭 체크!
    ↓
  Install Now
    ↓
python --version
    ↓
  ✅ Python 3.11.0
```

### 3단계: Git 설치
```
git-scm.com
    ↓
  다운로드
    ↓
  설치 (Next 계속 클릭)
    ↓
git --version
    ↓
  ✅ git version 2.40.0
```

### 4단계: 텔레그램 봇
```
텔레그램 앱
    ↓
@BotFather 검색
    ↓
/newbot 입력
    ↓
봇 이름 입력
    ↓
봇 사용자명 입력
    ↓
🔑 토큰 받기
    ↓
봇에게 메시지 보내기
    ↓
getUpdates로 채팅 ID 확인
    ↓
🆔 채팅 ID 받기
```

### 5단계: 지역 코드
```
new.land.naver.com
    ↓
원하는 동네 검색
    ↓
URL 보기:
https://...?cortarNo=1168010600&...
                    ↑↑↑↑↑↑↑↑↑↑
                    이 숫자!
    ↓
📍 지역 코드 복사
```

### 6단계: 프로그램 준비
```
PowerShell 열기
    ↓
cd "프로젝트 폴더"
    ↓
python -m venv venv
    ↓
.\venv\Scripts\Activate.ps1
    ↓
(venv) ← 이게 보이면 성공!
    ↓
pip install -r requirements.txt
    ↓
.env 파일 만들기
    ↓
토큰, 채팅ID, 지역코드 입력
    ↓
✅ 준비 완료!
```

### 7단계: 테스트
```
cd src
    ↓
python test_basic.py
    ↓
[OK] 5개 모두 통과?
    ↓
    YES!
    ↓
python main.py
    ↓
크롤링 시작...
    ↓
📱 텔레그램 알림!
    ↓
🎉 성공!
```

## 📝 필요한 정보 정리 표

| 항목 | 예시 | 어디서 구하나요? |
|------|------|------------------|
| **텔레그램 토큰** | `1234567890:ABC...` | @BotFather |
| **채팅 ID** | `987654321` | getUpdates API |
| **지역 코드** | `1168010600` | 네이버 부동산 URL |
| **거래 유형** | `A1,B1` | A1=매매, B1=전세 |

## 🎨 .env 파일 템플릿

```
┌─────────────────────────────────────────────┐
│ .env 파일 내용                               │
├─────────────────────────────────────────────┤
│                                             │
│ TELEGRAM_BOT_TOKEN=여기에_토큰              │
│ TELEGRAM_CHAT_ID=여기에_채팅ID               │
│ SEARCH_REGIONS=여기에_지역코드               │
│ TRADE_TYPES=A1,B1                           │
│                                             │
└─────────────────────────────────────────────┘

실제 예시:
┌─────────────────────────────────────────────┐
│ TELEGRAM_BOT_TOKEN=1234567890:ABCdef...     │
│ TELEGRAM_CHAT_ID=987654321                  │
│ SEARCH_REGIONS=1168010600                   │
│ TRADE_TYPES=A1,B1                           │
└─────────────────────────────────────────────┘
```

```

```
명령어: .\venv\Scripts\Activate.ps1

텔레그램 앱 확인

```
명령어: python main.py
✅ 성공: 크롤링 완료 + 텔레그램 알림
❌ 실패: 에러 메시지
```

## 🎯 가장 많이 하는 실수

### 실수 1: 체크박스 안 체크
```
Python 설치 시:
❌ [ ] Add Python to PATH
✅ [☑] Add Python to PATH  ← 이렇게!
```

### 실수 2: 가상환경 안 켬
```
❌ PS C:\...>
✅ (venv) PS C:\...>  ← 앞에 (venv) 있어야 해요!
```

### 실수 3: .env 파일 잘못 만듦
```
❌ TELEGRAM_BOT_TOKEN = "1234..."  (공백, 따옴표 있음)
✅ TELEGRAM_BOT_TOKEN=1234...      (공백, 따옴표 없음)
```

### 실수 4: 봇 START 안 누름
```
텔레그램에서:
❌ 봇 찾기만 함
✅ START 버튼 클릭 + 메시지 보내기
```

## 🎓 용어 설명

### Python (파이썬) 🐍
```
┌────────────────┐
│ 사람의 말      │
│   ↓ 번역       │
│ Python         │
│   ↓ 번역       │
│ 컴퓨터가 이해   │
└────────────────┘
```

### 가상환경 (venv) 🏠
```
┌─────────────────┐
│  내 컴퓨터      │
│  ┌───────────┐  │
│  │ 가상환경  │  │
│  │  ↓        │  │
│  │ 프로그램  │  │
│  │ 전용 공간 │  │
│  └───────────┘  │
└─────────────────┘
```

### API (에이피아이) 🔌
```
우리 프로그램  ←→  API  ←→  네이버
    "집 정보 줘!"  →  [요청]
    [응답]  ←  "여기 있어요!"
```

### 토큰 (Token) 🔑
```
┌────────┐
│  🔑    │  ← 집 열쇠처럼
│ 토큰   │     봇을 쓸 수 있게
└────────┘     해주는 열쇠
```
